A practical prototype for leadership teams to choose seven change readiness indicators, vote on current and desired state, and see where the biggest gaps and alignment issues are. It is simple by design, but already useful for dialogue and early testing.

This version is intentionally simple. It runs in one browser, stores data locally, and can generate a shareable snapshot link. For true team-wide live collaboration later, you would add a backend and user access.

How this first prototype works

You choose seven indicators from a library. Each team member can then add their own ratings for current state and desired state. The app calculates averages, gap size, and alignment spread.

Current state

How the team experiences the topic today.

Desired state

Where the team wants to be in a realistic next phase.

Gap

The distance between current and desired state.

Alignment

How similarly or differently people see the same reality.
